# 📖 Modern Cordel

This repository contains my solution to the **Modern Cordel** challenge from the **HTML5 & CSS3** course by **Professor Gustavo Guanabara (Curso em Vídeo)**.

The goal of this project was to practice modern CSS techniques while recreating the famous *Cordel Moderno* webpage.

## 🚀 What I practiced

* Semantic HTML5
* CSS3 styling
* Custom fonts
* Responsive layout
* Background images
* **Parallax scrolling effect**
* Text formatting and spacing
* CSS variables
